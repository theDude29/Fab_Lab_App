Pour lancer ce projet il vous faut installer node.js (https://nodejs.org/fr/about/) :
    https://nodejs.org/fr/download/ (Prenez une version LTS  autrement ca marche pas)

Ensuite il faut installer expo (https://expo.io/) :
    mettez ca dans un terminal -> $ npm install -g expo-cli

creer projer expo dans un dossier :
    mettez ca dans un terminal -> $ expo init Fab_Lab_App
    choisissez 'blank' comme type de projet
    supprimer tous les fichier present dans le dossier

cloner ce projet dans le repertoire ou le projet expo a été initialisé

mettez vous a la racine du projet : 
    mettez ca dans un terminal : $ cd [emplacement du projet]

installer les dependances : 
    mettez ca dans un terminal : $ npm install

lancer le serveur node js :
    mettez ca dans un terminal : $ npm start
    
scanner le QR Code et admirer le resultat ;-)
